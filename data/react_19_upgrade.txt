Rule: Ref passing in React 19
In React 19, you can now access ref as a prop for function components. Previously, you had to use the forwardRef higher-order component.

Legacy Pattern (React 18):
const MyInput = forwardRef((props, ref) => <input {...props} ref={ref} />);

Modern Pattern (React 19):
function MyInput({placeholder, ref}) { return <input placeholder={placeholder} ref={ref} />; }

Security Note: This change is part of the 2026 "Component Flattening" standard to reduce the attack surface for serialization-based RCEs like CVE-2026-23864.
